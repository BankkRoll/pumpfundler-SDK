# Introduction to Pumpfundler SDK

Welcome to the Pumpfundler SDK documentation! This powerful TypeScript library is designed to simplify your interaction with the PumpFun protocol and bundles on the Solana blockchain. Whether you're creating tokens, managing bonding curves, or integrating advanced features like MEV protection, Pumpfundler SDK has got you covered.

## What is Pumpfundler?

Pumpfundler is a revolutionary protocol on the Solana blockchain that enables the creation and management of tokens with customizable bonding curves, as well as efficient bundling of transactions. It provides a suite of tools for developers to build innovative DeFi applications with ease.

## Getting Started

To begin using the Pumpfundler SDK, follow these steps:

<Steps>
  <Step title="Install the SDK">
    Run the following command in your project directory:
    ```bash
    npm install pumpfundler-sdk
    ```
  </Step>
  <Step title="Configure your environment">
    Set up your Solana connection and wallet:
    ```typescript
    import { Connection, Keypair } from "@solana/web3.js";
    import { AnchorProvider, Wallet } from "@coral-xyz/anchor";

    const connection = new Connection("https://api.mainnet-beta.solana.com");
    const wallet = new Wallet(Keypair.generate());
    ```
  </Step>
  <Step title="Initialize the SDK">
    Create a PumpfundlerSDK instance:
    ```typescript
    import { PumpfundlerSDK, PumpfundlerConfig } from "pumpfundler-sdk";

    const config: PumpfundlerConfig = {
      connection,
      jitoFee: 1000000, // 0.001 SOL
      commitmentLevel: "confirmed",
      blockEngineUrl: "https://your-jito-block-engine-url.com",
      jitoAuthKeypair: "your-jito-auth-keypair-base58",
    };

    const provider = new AnchorProvider(connection, wallet, {});
    const sdk = new PumpfundlerSDK(provider, config);
    ```
  </Step>
</Steps>
